# 🔍 TrustHire – Fake Job & Internship Detector

TrustHire is an AI-powered web application that detects whether a job or internship posting is **fake or legitimate** using Natural Language Processing and Machine Learning.

---

## 🚀 Features

- Detect fake job postings using NLP
- TF-IDF + Logistic Regression model
- Scam keyword feature engineering
- Flask backend API
- React frontend UI
- Confidence score for predictions

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- NLTK
- Flask
- React.js
- HTML/CSS/JavaScript

---

## ⚙️ How it works

1. User pastes job description
2. Text is cleaned & vectorized
3. ML model predicts fake or legit
4. Result + confidence is displayed

---

## 📊 Model Performance

- Accuracy: ~97%
- Fraud Recall: ~88%
- F1 Score: ~0.68
